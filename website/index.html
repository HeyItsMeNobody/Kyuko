List of all Kyuko's commands
Required arguments look like this
Misc
k!say message make the bot say something!
k!ping says pong back
k!help brings you here
Admin commands
k!saychannel channelid message makes the bot say something in a channel
k!setwelcome message or reset set the welcome message (to mention user use user_mention)
k!setleave message or reset set the leave message (to mention user use user_mention)
Bot owner commands
k!whitelist serverid servername or DELETE serverid whitelists a server
k!setactivity type message sets the bots activity
